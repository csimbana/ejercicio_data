# Prueba Técnica - Data
Ejercicio de Data Engineer
En este repositorio se encuentra la resolución de una prueba técnica enfocada en Spark

El ejercicio incluye:

- Consultas SQL
- Procesamiento de datos con PySpark
  
Desarrollado en Google Colab.

## Cómo usarlo

Abrir el archivo `.ipynb` en Colab y ejecuta las celdas en orden.
